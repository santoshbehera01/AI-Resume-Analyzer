# AI Resume Analyzer

A modern React + Vite application for analyzing resumes against job descriptions using AI-powered ATS scoring and career coaching.

## Features

- **ATS Score Analysis**: Get detailed 0-100 scores showing resume compatibility with ATS systems
- **AI Career Coach**: Receive personalized feedback on resume sections and content
- **Multi-Role Matching**: Compare resumes against multiple job roles simultaneously
- **Skill Analysis**: Dynamic keyword extraction and matching
- **PDF Reports**: Download professional analysis reports
- **Authentication**: Secure user accounts with history tracking

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **UI**: Tailwind CSS, Radix UI components
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **AI**: Custom NLP algorithms for resume analysis
- **Deployment**: Production-ready SaaS architecture

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run development server: `npm run dev`
